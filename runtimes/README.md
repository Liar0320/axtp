# AXTP Runtime Implementations

Full runtime implementations have been moved to dedicated repositories:

- `axtp-c-runtime`
- `axtp-cpp-runtime`
- `axtp-flutter-runtime`
- `axtp-ts-runtime`
- `axtp-python-runtime`
- `axtp-mock-server`

This repository remains the AXTP Spec source of truth.

Runtime repositories declare their AXTP Spec compatibility via
`AXTP_SPEC.lock.yaml`. Do not treat runtime repositories as protocol sources.
